# Androguard

**Load an APK in a shell:**
```
androlyze.py APK
```

**See certificate:**
```
androsign.py --all --show APK
```

**Show package information:**
```
androguard apkid APK
```

**Show the manifest:**
```
androguard axml APK
```

**Decode compiled resources:**
```
androguard arsc app.apk
```
